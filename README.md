# Vibe Front-End Tuner

一个给 vibe coding / AI 前端生成流程使用的小型视觉调参工具。

## 项目想法

在让 AI 写网站、写前端界面时，我们经常会遇到一个问题：

> 人描述出来的效果，和 AI 实际生成出来的页面效果，差距很大。

比如我们说“更高级一点”“更像 Claude 一点”“间距舒服一点”“卡片轻一点”“标题不要那么大”，AI 往往会理解得很模糊，生成结果也容易漂移。

这个工具的作用就是把这些模糊的审美感觉，转成一组可以复制给 AI 的前端参数。

你可以先在工具里自己拖动滑杆，调整页面宽度、区块间距、卡片间距、字号、行高、圆角、阴影、描边、颜色温度等视觉参数。右侧会实时预览产品页、仪表盘和表单三种常见前端界面。

调到舒服之后，直接复制生成的 CSS Token 或 Prompt 参数，再发给 ChatGPT、Claude Code、Codex、Cursor、Trae 等 AI 编程工具，让 AI 更稳定地理解你想要的前端效果。

## 它解决什么问题

- AI 生成的 UI 和用户脑海里的画面差距过大
- “高级感”“留白”“质感”“克制”这类描述太抽象
- 每次让 AI 改前端，字号、间距、圆角、阴影都会漂移
- 用户想把主观审美转成可复用、可复制、可迭代的参数

## 怎么使用

1. 打开 `index.html`
2. 拖动左侧参数滑杆
3. 在右侧切换预览：产品页 / 仪表盘 / 表单
4. 调到满意后，复制 CSS Token 或 Prompt 参数
5. 把参数发给 AI，让它按这组视觉参数修改你的真实项目

## 适合谁

- 经常用 AI 写前端的人
- 做个人网站、作品集、仪表盘、落地页的人
- 想把“感觉不对”变成明确参数的人
- 想搭建自己视觉 token / design system 的人
